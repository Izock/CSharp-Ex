\# C# Learning Journey



Repositorio donde documento mi aprendizaje de C# y .NET.



\## Objetivo



Aprender los fundamentos de C# mediante ejercicios prácticos, proyectos pequeños y retos de programación.



\## Temas estudiados



\- Variables y tipos de datos

\- Operadores

\- Condicionales (if, else, switch)

\- Ciclos (for, while, foreach)

\- Métodos

\- Arrays y colecciones

\- Programación orientada a objetos

\- LINQ

\- Manejo de excepciones

\- Archivos y directorios



\## Estructura del proyecto



```text

Exercises/

├── HelloWorld

├── Variables

├── Loops

├── Methods

└── Classes

```



\## Requisitos



\- .NET SDK 8.0 o superior

\- Visual Studio 2022 o Visual Studio Code

```bash

dotnet run

```
